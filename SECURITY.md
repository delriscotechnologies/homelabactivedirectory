# Security policy

This repository documents an isolated training sandbox. Selected screenshots contain disposable accounts, private addresses, identifiers, and credentials created specifically for the exercise. They were used only inside this lab and were never used or reused on personal, production, or external systems. Never reuse these values or publish personal or production credentials here.

Report security concerns through GitHub's private vulnerability reporting option in the repository's **Security** tab when it is available. Do not post real credentials or private infrastructure details in a public issue.
